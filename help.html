User Guide

Welcome to My Notes! This is a detailed guide to help you use the application.

Create a new note

Click the "Create New" button at the top right to open the note creation form. Fill in the title, content, select a category and color, then click "Save Note".

Find notes

Use the search bar at the top of the page to find notes by title or content. Results will display instantly.

Filter by category

Use the filter buttons (Pet, Security, Work, Personal) to view notes by specific category.

Edit & Delete

Hover over a note to reveal the edit (pen icon) and delete (trash icon) buttons.

Dark Mode

Click the moon/sun icon at the top right to toggle between light and dark mode.

Tips for Using

  • Use different colors to easily categorize notes
  • Write concise and clear titles for easier searching
  • Use categories to organize notes by topic
